Changes Implemented:

Header: The rounded border is now restricted to the text area (Title, Address, Phones). The "REPAIR SLIP" badge and "5581" are outside to the right, matching the original image layout.

Date & Tel No Alignment: I have structured the input section so "Tel No" sits directly below "Date" in a dedicated right-hand column, ensuring perfect vertical alignment.

Rounded Rows (Pills):

Fault / Model is now in its own isolated rounded border.

IMEI / Make is now in its own isolated rounded border.

The Description & Total section is in a separate block below.

Vertical Line & Alignment: Even though the rows are separated, I used fixed percentage widths (78% Left / 22% Right) for all sections. This ensures the vertical line appears to run straight down through the gaps, and Model/Make/Total are positioned on the left side of that line.

        function savePDF() {
            const element = document.getElementById('repair-slip');
            const opt = {
                margin: 0,
                filename: 'repair-slip.pdf',
                image: { type: 'jpeg', quality: 1 },
                html2canvas: { scale: 2 },
                jsPDF: { unit: 'px', format: [890, 800], orientation: 'portrait' }
            };
            html2pdf().set(opt).from(element).save();
